# Metal-Trader-IQ
A Decision-Making Digital Twin designed for a mining company. The project integrates real-time LME (London Metal Exchange) data with physical inventory management. It calculates automated gross/net margins, incorporating local taxation (Octroi de Mer), and provides a Power BI dashboard for real-time inventory and profitability analysis.

# Metal-Trader IQ : Jumeau Numérique pour Entreprise Minière

## 🚀 Présentation
Projet de jumeau numérique décisionnel conçu pour optimiser l'arbitrage financier et la gestion des stocks dans une entreprise minière en Guadeloupe. 
Ce projet croise les données mondiales (LME) et les contraintes logistiques locales (Octroi de mer).

## 🛠️ Stack Technique
- **Data Analytics:** Microsoft Power BI
- **Language:** DAX (Data Analysis Expressions)
- **Modélisation:** Digital Twin Architecture

## 📈 Indicateurs Clés (DAX)
- Marge Arbitrage Brute
- Marge Arbitrage Nette (ajustée fiscalité)
- Valeur de Stock en temps réel
